# Employee Management System

A full-stack Employee Management System built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

## Features

- Add Employee
- View Employee List
- Delete Employee
- MongoDB Database Integration
- Responsive User Interface

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Create a `.env` file

```
MONGO_URI=mongodb://localhost:27017/employeedb
PORT=3001
```

4. Run the application

```bash
node server.js
```

5. Open:

```
http://localhost:3001
```
